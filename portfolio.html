    Welcome to my workspace. Here you will find a few of my projects along with a respective description. Ideally I prefer to design and develop custom websites. My main focus is to make each website unique and robust. I try to use CSS(3) for presentation and Javascript for functionality. Web Development (and Design)

Betafish is a website currently under development that I was hired to design and build to advertise a mobile fishing application. Built from scratch, I used SASS as my preprocessor for CSS and utilized Javascript/JQuery as needed.

ChristinaErichsen.com is a website I built and designed from the ground up as a portfolio for a real estate agent. It was built with SASS and includes custom JQuery animations and a Vegas2 background slideshow. View it live by clicking here

Hysucat is a homepage that I built from scratch based off of a design in a .psd file. This page was developed in less than a day from scratch using SASS and utilizes JQuery, Animate.css and Parallax.js.
